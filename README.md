# Transdutor em ruby
esse repositório foi feito pra estudar linguagens formais e autômatos. Foi implementado um transdutor de números romanos para numerais arábicos. Por exemplo:

`XVIII = 18`

a implementação foi feita usando a linguagem ruby por recomendação do professor.

a modelagem, especificações e formalismos do projeto estão dentro da pasta `docs` na raiz do repositório.
